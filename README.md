# Hello Rails React

> Set up a "Hello world" project as one app with Webpack


## Built With

- ROR
- React.js
- Webpack

## Getting Started

- Copy this repository
- do bundle install
- Configure database.yml
- Create database, run migrations and seed file
- Run rails server

## Prerequisites
* Ruby V3+, 
* Rails V7+
* Postgresql
